== README

Skills/Gems I used:

* [BH - BootStrap Helper Gem](https://github.com/Fullscreen/bh)

* Bootstrap

* FactoryGirl

* Pry

* Selenium-Webdriver

* Integrated [Flatty Theme](http://blacktie.co/2013/12/flatty-app-landing-page/)

* View the app deployed at Heroku - [yaurls-tdd.herokuapp.com](yaurls-tdd.herokuapp.com)
